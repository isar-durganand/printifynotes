# Printify Notes

Convert dark background PDF notes to printable format. Perfect for Physics Wallah, Unacademy, Vedantu, and other coaching platform notes.

## Features

- Convert dark PDF backgrounds to light/printable format
- Save ink when printing study notes
- 100% private - no file uploads, everything processed in browser
- Free to use with no limits

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```

## Building for Production

```sh
npm run build
```
